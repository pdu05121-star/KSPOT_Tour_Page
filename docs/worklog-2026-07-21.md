## 2. 춘천·강릉 상세페이지 — KSPOT Travelog 전환 (틀만)

[[design-system-addendum-tour-content]]에서 미결정 사항으로 남아있던 "기존 이커머스형 페이지도 Travelog 서브 브랜드로 전환할지"에 대해, 이번 세션에서 춘천·강릉 페이지에 한해 전환을 진행함 (체크박스 업데이트는 해당 문서에 반영).

**한 것**:
- `ChuncheonTour.tsx`, `GangneungTour.tsx`를 가짜 별점·후기·신청인원·카운트다운·Supabase 리드폼이 있던 이커머스형에서, `SuwonTour.tsx`와 동일한 디자인 틀(스크랩북 톤, SPOT 배지, 챕터 구성)로 교체

**하지 않은 것 (중요 — 미작업 상태)**:
- 실제 왕복 교통시간(열차/버스/KTX)이 확정되지 않아, 판정을 임의로 GO/CARE로 매기지 않고 `◐ DRAFT` 상태로 정직하게 표시 (수치 조작 금지 원칙 — Hard Gate 로직과 동일한 정신)
- 콘텐츠(스팟 설명·주소·팁)는 기존 이커머스형 페이지 문구를 그대로 재사용 — 사실 검증 안 됨
- 사진은 전부 임시 placeholder
- 다국어(영/일/중) 미적용, 한국어만
- 구글 마이맵 저장 링크 없음 (아직 안 만들어짐)
- 목록 페이지(`/tour`)에서는 계속 잠금 카드로 유지 — 실작업 전까지 사용자에게 노출 안 함

**결정 이유**: "판정보다 먼저" 원칙상, 왕복 판단이 안 된 코스를 GO로 보여줄 수 없음. 틀만 먼저 맞추고 실제 콘텐츠 작업(교통시간 확정, 번역, 사진, 마이맵)은 별도 트랙으로 이어감.

## 3. 랜딩 페이지 CTA·투어목록 인트로 카피 통일

**배경**: 랜딩페이지 CTA 버튼들이 목적지가 제각각(`/tour/suwon` 직행 / 페이지 내 앵커 / 구글폼)이고 문구도 통일감이 없었음. 투어 목록 페이지 인트로 카피도 "숨은 명소 추천" 톤이라 랜딩의 "판단 서비스" 포지셔닝과 결이 어긋났음.

**결정**:
- 탐색성 CTA(nav/히어로 2번째 버튼/무드브레이크/투어 섹션) 4곳 → 전부 `/tour`(목록페이지)로 통일, 문구도 **"안심 코스 보기"**로 통일
- 설문 CTA("원하는 코스 받아보기 →" → 구글폼)와 문의 CTA("팀에게 문의하기" → mailto)는 변경 없음
- 최종 CTA 섹션의 "수원 코스 먼저 체험하기" 버튼 제거 (중복 동선이라 삭제)
- 투어 목록 페이지 인트로: 기존 "한국에 이런 곳이 있다고?"라는 발견형 훅은 살리고, 뒤에 "가도 괜찮은 곳인지, 저희가 먼저 확인했어요."를 붙여 판단 서비스 메시지로 연결 (4개 언어 모두 반영, AI 초벌번역 — 원어민 검수 필요)

**참고**: 이 결정 과정에서 "걱정마세요" 버전을 먼저 적용했다가, 앞 문장(감탄/발견)과 논리적으로 안 이어진다는 피드백으로 "확인했어요" 버전으로 재수정함. (커밋 3개: `6312c95b` → `c66bf6a7` → `bdd28b2f`)

## 6. 수원 코스 "구글 지도에 저장하기" — 서울역 핀 추가 결정 (실행 대기)

**배경**: Chapter 4 타임테이블에 서울역 출발(09:00)·서울역 도착(18:22) 행을 추가한 뒤, "구글 지도에 저장하기" 버튼이 가리키는 마이맵(`mid=1aWeZoTginTKzpvtjmGTbihFzFYPssus`)에는 여전히 수원 내 스팟 7곳만 있고 서울역이 빠져 있다는 지적이 나옴. 처음엔 "걷기 편한 확대(zoom) 유지"를 이유로 지금처럼 7곳만 유지하자고 제안했으나, "KSPOT의 차별점은 서울↔수원을 무사히 왕복할 수 있는지 판단하는 것"이라는 반론에 따라 결정을 뒤집음.

**결정**: 서울역을 이 마이맵에 핀으로 추가한다. 왕복 판단이 KSPOT의 핵심 차별점이므로, 저장되는 지도 자체에도 그 왕복의 시작·끝이 보여야 한다는 논리를 우선함.

**미실행 상태 — 계정 접근 필요**: 이 마이맵은 코드가 아니라 mymaps.google.com에 만들어둔 구글 계정 리소스라 Claude Code가 직접 편집 불가 (계정 로그인이 필요한 작업은 대행 금지 원칙). 아래 절차로 계정 접근 권한 있는 팀원이 직접 추가해야 함:
1. mymaps.google.com 접속 → "선재업고튀어 타임슬립 투어" 지도 열기 → 편집 모드
2. 기존 레이어(`수원_선재업고튀어_코스.csv`)에서 "서울역" 검색 → 지도에 추가
3. 리스트 맨 앞(01번 앞)에 오도록 이름을 "00 서울역" 형태로 정렬

**후속 작업**: 위 마이맵 편집이 완료되면, [SuwonTour.tsx](../src/app/pages/SuwonTour.tsx)의 `saveNote` 카피("스팟 7곳이 담긴 지도예요")를 "서울역 출발부터 7개 스팟, 왕복까지 담긴 지도예요" 식으로 4개 언어 모두 업데이트할 것.
